Cycle Nearby

The "Cycle Nearby" application will identify features relevant to cyclists in Austin, TX that are closest to you. The features it knows about are:

  • bicycle thefts
  • bicycle-auto collisions
  • police station
  • fire station
  • post office
  • library
  • moon tower (huh?)

When you first start the application it will try to determine where you are. Depending on your web browser settings, you may be prompted to allow your location to be identified. Once you have been located, it will draw a map showing your position and the nearby features.

Here are some things you can do with the map:

  • Click at a location on the map to see what features are nearby.
  • Drag the red "you are here" marker around, to explore a different area.
  • Type an address in the search field at the top of the screen to map that location.

Powered by Open Source and Open Government

In January 2012, the City of Austin released the data.austintexas.gov open government data portal. This application is built on publicly available data provided there. At this time, there are only a limited number of data collections available, so only a small set of features are supported here. What else would you like to see? How about driver license offices? Public wi-fi access points? If the City releases that information, I'll add it to the application.

This application builds on "Find It Nearby" (see below), modified by Jeffrey L. Taylor to include features for cyclists. The modifications were started by the "Bike App" team at the "Code for America Hack-a-thon", September 2012. The source code is available at https://github.com/AustinBlues/findit.

Find It Nearby was written by Chip Rosenthal, with support from the "Find It" team from the "Code for America Hack-a-Thon", February 2012. The Find It Nearby source is available at: https://github.com/chip-rosenthal/findit
