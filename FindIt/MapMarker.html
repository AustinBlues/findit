FindIt::MapMarker

A graphic icon to be placed on the map as a marker.

For a list of map icons available at Google, see: sites.google.com/site/gmapsdevelopment/

new(url, params = {}) click to toggle source

Construct a new MapMarker.

Parameters:

  • url – The URL of the graphic.

  • params – Parameters for the marker.

Returns: A FindIt::MapMarker instance.

The following params are supported.

  • :height => Integer – Height of the graphic in pixels. (required)

  • :width => Integer – Width of the graphic in pixels. (required)

# File lib/findit/map-marker.rb, line 31
def initialize(url, params = {})
  raise("required parameter \":height\"  missing") unless params.has_key?(:height)
  raise("required parameter \":width\"  missing") unless params.has_key?(:width)
  @url = url
  @height = params[:height].to_i
  @width = params[:width].to_i
end
