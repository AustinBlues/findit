A graphic icon to be placed on the map as a marker.

new(url, params = {}) click to toggle source

Construct a new MapMarker.

Parameters:

  • url — The URL of the graphic.

  • params — Parameters for the marker.

Returns: A FindIt::MapMarker instance.

The following params are supported.

  • :height => Integer — Height of the graphic in pixels. (required)

  • :width => Integer — Width of the graphic in pixels. (required)
