FindIt::RakeDefs

This module can be included in “Find It” Rakefile’s that manage data sources.

They typically are found in: findit/local/locality/data/dataset

Constants

DB

Connection to the PostGIS/Postgresql database.

This connects as user “postgres” and no password. This will work if you run this as a user that has permissions to access the database in the “postgres” role.

The “findit” role is not sufficient, because this needs to make entries into the GIS tables.

SHP2PGSQL

Path to shp2pgsql utility.

I need this because my Ubuntu system does not put it in a public bin directory.

def db_load_shapefile(table, shapefile, srid)    
  IO.popen([SHP2PGSQL, "-s", srid, shapefile, table]) do |pp|
    save_lines = []
    pp.each_line do |line|
      next if line =~ /^--/
      save_lines << line.strip!
      next unless line =~ /;$/
      db_execute(save_lines.join(' '))
      save_lines = []
    end
  end  
end
